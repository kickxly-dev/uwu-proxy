Upload Guide

Zip must contain an index.html at the root level
All assets (JS, CSS, images) should be relative paths inside the zip
Max zip size is 50MB — compress assets where possible
The slug becomes the game URL: /games/YOUR-SLUG/
Icon URL is optional — use a direct image link for best results

Uploaded games are stored on Netlify's file system. Connect a database for persistent storage and metadata editing.
